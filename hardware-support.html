This guide explains how to verify hardware compatibility, install drivers, and troubleshoot hardware-related issues in Linux.

Check Hardware Compatibility

Before installing Linux, ensure that your hardware is compatible. Visit the Linux Hardware Database: linux-hardware.org.

Run the following command to generate a hardware report:

sudo lshw -short

Installing Drivers

Graphics Drivers

For optimal graphics performance, install the appropriate drivers:

  • NVIDIA: Use the proprietary drivers:
    • sudo apt install nvidia-driver-XXX

    (Replace XXX with the recommended version.)

  • AMD: The open-source drivers are included in most distributions.
  • Intel: Drivers are included by default.

Wi-Fi and Network Drivers

Check for Wi-Fi drivers if your adapter isn't working:

sudo apt install firmware-linux-nonfree

For Realtek Wi-Fi adapters, additional drivers may need to be compiled from the source.

Peripheral Devices

Ensure peripherals like printers, scanners, and USB devices work by installing required packages:

sudo apt install hplip

For HP printers, or consult your device's documentation.

Troubleshooting Hardware Issues

Display Issues

If you encounter screen tearing or resolution problems, configure your graphics drivers or use xrandr:

xrandr --output HDMI-1 --mode 1920x1080

Audio Problems

Restart the audio service if sound isn’t working:

sudo systemctl restart alsa

Unrecognized Hardware

If a device isn’t recognized, check system logs for errors:

dmesg | grep error
