Linux Installation

This guide explains how to install Linux on a physical machine.

Step 1: Download a Linux Distribution

Visit the official website of a Linux distribution like Linux Mint, Ubuntu, or Fedora.

Download the ISO file for your system's architecture (64-bit or 32-bit).

Step 2: Create a Bootable USB

To install Linux, you need to create a bootable USB stick. Use tools like:

  • Rufus (Windows)
  • Balena Etcher (Cross-platform)
  • `dd` command (Linux)

Command to create a bootable USB on Linux:

sudo dd if=linux-distro.iso of=/dev/sdX bs=4M status=progress

Replace `linux-distro.iso` with the path to your ISO file and `/dev/sdX` with your USB device.

Step 3: Boot from USB

Restart your computer and access the boot menu (usually by pressing F12, Esc, or Del during boot).

Select your USB stick and boot into the Linux live environment.

Step 4: Install Linux

Follow the on-screen instructions:

  1. Select your language and keyboard layout.
  2. Choose "Install" or "Try & Install".
  3. Partition your disk (automatic or manual).
  4. Set up a user account and password.

Wait for the installation to complete, then reboot the system.

Step 5: Post-Installation Setup

After installation, update your system using the terminal:

sudo apt update && sudo apt upgrade

Install basic software like web browsers, text editors, and media players.
